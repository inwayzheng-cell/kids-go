# 🎨 小英雄圍棋 (Little Hero Go) - 9x9 呼吸能量版

這是一個專門為 **10 歲以下小朋友**設計的數位圍棋啟蒙工具。透過直觀的「視覺化氣」與「能量咬痕」效果，讓孩子在玩耍中自然理解圍棋最核心的生存規則。

## 🌟 遊戲特色

- **呼吸光環 (Aura)**：每顆小怪獸（棋子）都有自己的氧氣圈。黑棋是紫色光環，白棋是綠色光環。
- **動態咬痕 (The Bite)**：當敵我相鄰時，光環會出現缺口，象徵「氣」被對方吃掉了，視覺化呈現氣的抵銷。
- **禁著點警告**：如果不小心點到會讓小怪獸窒息的「黑洞（禁著點）」，棋盤會發出震動提醒。
- **全自動縮放**：支援手機、平板與電腦，畫面會根據裝置大小自動調整完美比例。

## 🕹️ 如何開始玩？

1. 直接開啟你的 GitHub Pages 網址（例如：`https://inwayzheng-cell.github.io/kids-go/`）。
2. **黑棋 (😈)** 先行，**白棋 (😇)** 隨後。
3. 觀察光環：
   - **綠/紫光環**：安全，呼吸順暢。
   - **光環閃爍**：危險！只剩下最後一口氣了。
   - **光環消失**：小怪獸被抓走囉！

## 🛠️ 技術細節

- **純前端實作**：僅使用單一 HTML 檔案，包含 CSS 繪圖與原生 JavaScript 邏輯。
- **無依賴庫**：不需安裝任何套件，隨開即玩。
- **響應式佈局**：使用 CSS Flexbox 與百分比單位 (%) 達成跨裝置適配。
